# BogerdPro Design System

## Principios

- Usabilidad antes que espectáculo: encontrar productos en menos de 3 clics.
- Rendimiento antes que ornamento: animaciones cortas, pocos elementos client-side e imágenes controladas.
- Mantenibilidad antes que excepciones visuales: tokens, componentes pequeños y patrones repetibles.
- Idioma principal: castellano. Idiomas previstos: catalán, inglés y neerlandés.

## Tokens

- Colores: `#F7F5F0` fondo, `#FFFFFF` superficie, `#151515` texto, `#62615D` texto secundario, `#E7E2D8` línea, `#3572B8` acento principal BogerdPro, `#245A96` acento oscuro.
- Tipografía: stack de sistema tipo Inter, sin fuente remota para proteger Core Web Vitals.
- Radius: 8px en controles, 12px en cards principales.
- Sombras: suaves y funcionales, solo para elevar elementos interactivos.
- Movimiento: 150-250 ms, fade/slide/scale ligero.

## Componentes

- Header con mega menú: acelera el descubrimiento de categorías B2B.
- Buscador visual: acción primaria para catálogos grandes.
- Filtros laterales: facetas claras sin cargar la parrilla.
- Cards de producto: imagen, categoría, referencia, stock y CTA visibles.
- Skeletons y empty states: reducen incertidumbre y mantienen el ritmo visual.
- Carrito flotante: acción siempre accesible sin interrumpir la navegación.
